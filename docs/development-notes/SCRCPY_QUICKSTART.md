# scrcpy 監看設備 - 快速開始

## 🎯 一句話說明

**在設備卡片的選單中點擊「📺 監看設備」，即可開啟視窗即時查看和控制 Quest 設備！**

---

## ⚡ 快速開始（3 步驟）

### **步驟 1：安裝 scrcpy**

```bash
# macOS
brew install scrcpy

# Linux (Ubuntu/Debian)
sudo apt install scrcpy

# Windows
# 下載：https://github.com/Genymobile/scrcpy/releases
```

### **步驟 2：重新啟動應用**

```bash
cd /Users/jinyaolin/QQquest
./run.sh
```

### **步驟 3：啟動監看**

1. 進入「📱 設備管理」頁面
2. 找到在線設備（🟢）
3. 點擊「**⋮**」→「**📺 監看設備**」
4. ✅ 視窗自動彈出！

## ⚠️ 注意事項

### **必須在線**
- ❌ 離線設備無法監看
- ✅ 在線設備（🟢）才能使用

### **需要安裝 scrcpy**
- 點擊「監看設備」如果沒反應
- 很可能是 scrcpy 未安裝
- 執行 `brew install scrcpy`（macOS）

### **視窗獨立**
- scrcpy 視窗是獨立進程
- 關閉 QQQuest 不會關閉監看視窗
- 可以同時開啟多個視窗

## 🐛 問題排除

### 問題：點擊沒反應

**解決**：
```bash
# 檢查 scrcpy 是否安裝
scrcpy --version

# 如果未安裝
brew install scrcpy  # macOS
```

### 問題：視窗立即關閉

**解決**：
1. 確認設備在線（🟢）
2. 嘗試重新連線設備
3. 檢查 Quest 的 USB 調試授權

### 問題：畫面延遲

**解決**：
- 使用更好的 WiFi 網路
- 靠近 WiFi 路由器
- 考慮使用 5GHz WiFi
