## 🎯 常見使用場景

### 場景 1：課程前設備檢查
1. 打開設備管理頁面
2. 檢查所有設備電量（> 50%）
3. 檢查溫度（< 35°C）
4. 確認設備清醒
5. ✅ 準備就緒

### 場景 2：發現異常設備
1. 卡片顯示 🔴 紅色指標
2. 點擊「⋮」選單
3. 根據問題採取行動：
   - 電量低 → 充電
   - 溫度高 → 散熱/休息
   - 開機久 → 重啟

### 場景 3：批量管理
1. 一眼掃描所有設備卡片
2. 快速識別顏色指標
3. 優先處理紅色/黃色設備
